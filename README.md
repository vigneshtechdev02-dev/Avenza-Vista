# Avenza Vista - Travel Website

Avenza Vista is a modern travel and tourism website designed using **HTML5** and **CSS3**. This project focuses on creating a clean, responsive, and user-friendly interface that helps users explore travel destinations and vacation packages. The website features an attractive layout with well-organized sections, providing an engaging browsing experience for travelers.

## Features

* Responsive navigation bar
* Main banner with call-to-action
* Popular Destinations section
* Travel Packages section
* Activities showcase
* Why Choose Us section
* Travel Blog & Tips
* About section
* Modern Footer with contact and social links
* Clean, responsive layout using CSS Flexbox

## Technologies Used

* HTML5
* CSS3
* CSS Flexbox
* Google Fonts
* Font Awesome Icons

## Website Sections

* Home
* Destinations
* Packages
* Activities
* Choose Us
* Blog & Tips
* About

## Purpose

This project was created to improve my frontend web development skills by building a modern travel website with a professional design and responsive layout. It demonstrates my understanding of HTML, CSS, Flexbox, website structure, and UI design principles. In the future, I plan to add JavaScript functionality, booking features, animations, and backend integration to transform it into a fully functional travel booking platform.
